### ❌ Modules Aren't Activating

**Cause:** Key conflict or GUI rendering issue.

- Try resizing the game window or toggling fullscreen mode.
- Re-execute the script without changing tabs rapidly during initialization.

### ❌ Script Crashes Mid-Session

**Cause:** Memory conflict with another running script.

**Fix:**
```bash

# Do not run multiple automation scripts simultaneously
```
- Uninject completely, close the game, relaunch, and inject fresh.

### ❌ Config Not Saving Between Sessions

**Cause:** Write permissions issue on config path.

**Fix (Windows):**
```bash
icacls "%appdata%\newvape" /grant %username%:F /T
```

**Fix (macOS):**
```bash
chmod -R 755 ~/Library/Application\ Support/newvape/
```

---

### ❌ Fly Module Detected / Kicked

**Cause:** Using fly at speeds above server threshold.

**Fix:** Reduce `fly_speed` to `1.0` or below. Use **Speed Boost** instead of Fly for horizontal movement where possible.
